## Contributing

Skills live in tiers â€” add yours to the right place:

| Tier | Path | Governance |
|------|------|------------|
| **PostHog** | `skills/posthog/` | Auto-generated by CI â€” don't edit |
| **Team** | `skills/team/` | PostHog team, PR to #team-docs-and-wizard |
| **Community** | `skills/community/` | Anyone, PR reviewed by maintainers |

To add a skill, create a directory with a `SKILL.md` containing YAML frontmatter (`name` + `description`) and open a PR. See the [skill format spec](spec/skill-format.md) for details.
